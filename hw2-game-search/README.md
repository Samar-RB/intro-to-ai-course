# Homework 2 – Game Search (Minimax & Alpha-Beta)

## Overview
This assignment focuses on solving a two-player game (IsoKnight) using adversarial search algorithms.

The project includes implementations of Minimax and Alpha-Beta pruning, along with heuristic evaluation functions for efficient decision-making in large game trees.

## Topics
- Minimax algorithm
- Alpha-Beta pruning
- Game tree search
- Heuristic evaluation functions
- Adversarial search

## Files
- `minimax_isoKnight.py` – Minimax implementation
- `alpha_beta_isoKnight.py` – Alpha-Beta pruning
- `heuristic_alpha_beta_isoKnight.py` – Alpha-Beta with heuristics
- `heuristics.py` – evaluation functions
- `game_state.py` – game representation
- `game_engine.py` – game logic
- `player_agent.py` – player decision logic

## Skills Demonstrated
- Adversarial search algorithms
- Optimization using pruning
- Heuristic design
- Game modeling in AI
- Efficient decision-making
