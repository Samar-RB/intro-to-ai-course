# Introduction to Artificial Intelligence

AI | Python | Reinforcement Learning | Search Algorithms | LLM Agents

---

## Overview
This repository contains coursework and projects from an Introduction to Artificial Intelligence course.

It covers core AI topics including search algorithms, game playing, planning, reinforcement learning, and modern agent-based systems using large language models (LLMs).

---

##  Topics Covered
- State-space search (A*)
- Adversarial search (Minimax, Alpha-Beta)
- Automated planning (PDDL)
- Reinforcement Learning (Q-Learning, Value Iteration)
- Agentic AI and LLM-based systems

---

##  Repository Structure

### 🔹 Homework 1 – Search & Heuristics
Implementation of search algorithms and heuristic functions for solving a blocks world problem.

### 🔹 Homework 2 – Game Search
Adversarial search using Minimax and Alpha-Beta pruning for a two-player game.

### 🔹 Homework 3 – Planning (PDDL)
Modeling and solving planning problems using PDDL and automated planners.

### 🔹 Homework 4 – Agentic AI
Multi-agent system using LLMs to solve problems, compare results, and act as a judge.

### 🔹 Homework 5 – Reinforcement Learning
Implementation of Q-Learning and Value Iteration for solving MDP problems.
